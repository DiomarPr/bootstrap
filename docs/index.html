Globo Bootstrap

O Bootstrap da Globo.com, baseado no Twitter Bootstrap. Desenvolvido para o desenvolvimento rápido dos componentes de interface. Os componentes vão desde HTML, CSS e plugins javascript para unificar e facilitar o desenvolvimento dos produtos da Globo independente da plataforma

Feito para todos, em todo lugar

Precisa de mais razões para usar o Globo Bootstrap?

Contribuição

O GloboCMA Bootstrap foi construído para incentivar a colaboração entre times e unificar as interfaces dos componentes visuais da Globo.com.

Para todos os níveis

Construído para facilitar o desenvolvimento de interfaces, independente da linguagem utilizada, o Globo Bootstrap foi construído para acabar de vez com o retrabalho do front-end dos elementos de interface de toda a globo.com

Suporte

Construído otimizado para os browsers mais modernos, o Globo Bootstrap foi feito para funcionar em várias plataformas, inclusive IE7 e na sua segunda versão foi otimizado para suporte a mobile usando o responsive design.

Grid de 12 colunas

Grid não é tudo, mas ter um grid flexível e bem projetado irá tornar o desenvolvimento mais simples

Responsive design

Com o Bootstrap 2, ele se tornou total responsive. Todos os componentes foram escalados de acordo com uma ampla variedade de resoluções e devices para prover uma experiência consistente, independente do dispositivo.

Guia de estilos

Incentivo para o uso de uma prática que ajuda bastante no desenvolvimento e componentização, com uma documentação compreensiva e prática, baseado em exemplos de código, de forma orgânica e intuitiva.

Orgânico

Dezenas de componentes visuais prontos para usar, com consistência e nos padrões de qualidade da globo.com. Novos componentes podem ser adicionados com a necessidade, mas tudo estará em um só lugar!

Plugins de jQuery

Plugins jQuery construídos prontos para serem usados de forma extensiva. Desta forma o Globo Bootstrap trará vida a seus projetos

Usando LESS

Com o less podemos criar de forma automatizada todos os CSS's necessários para o projeto de forma organizada, usando variáveis e composição em cascata

HTML5

Desenvolvido para suportar elementos HTML5 e sintaxe

CSS3

Elementos são melhorados progressivamente de acordo com a capacidade do dispositivo

Caminhando para o Open-source

Projeto piloto que poderá futuramente ser open-source, já que somos uma variação do Bootstrap do twitter

Feito pelo Twitter, usado na Globo.com

Uma simples idéia desenvolvida pelo Twitter e aplicada para resolver as necessidades da globo.com na inconsistência entre plataformas que deveriam ser iguais quando se fala em interface.
