O grid padrão disponível como parte do Bootstrap é um grid de 940px, de 12 colunas

Existem três variações de grid responsive para vários devices e resoluções: phone, tablet, e telas maiores de desktop

  <div class="span8">...</div>

Colunas aninhadas

Com o grid estático (não fluido) do Bootstrap, aninhamento é bem fácil. Para aninhar o conteúdo, simplesmente adicione um novo .row e blocos de colunas .span* dentro de um span existente .span*.

Exemplo

Nível 1 da coluna
Nível 2
Nível 2 
<div class="row">
  <div class="span12">
    Nível 1 da coluna
    <div class="row">
      <div class="span6">Nível 2</div>
      <div class="span6">Nível 2</div>
    </div>
  </div>
</div>

Customização do grid

Variáveis Valor padrão Descrição
@gridColumns 12 Número de colunas
@gridColumnWidth 60px Largura de cada coluna
@gridGutterWidth 20px Espaço negativo entre colunas
@siteWidth Somas computadas de todas as colunas e margens Contagem de número de colunas e gutters estão em .container-fixed()

Variáveis em LESS

Através das variáveis é possível customizar o grid de 940px padrão, documentado acima. Todas variáveis para o grid estão armazenadas em variables.less

Como customizar

Modificar o grid significa modificar as três variáveis @grid* e recompilando o Bootstrap Mudando as variáveis do grid e use uma das opções de compilar. Existem quatro maneiras documentadas para compilar

Continuando responsive

A customização do grid somente funciona no seu padrão, ou seja, o grid de 940px. Para manter os aspectos responsive do Bootstrap, você também terá que customizar os grids em responsive.less.

Layouts Templates básicos para criar páginas

Layout fixo

O layout padrão e simples de 940px, centralizado para qualquer website que suporte um simples <div class="container">.

<body>
  <div class="container">
    ...
  </div>
</body>

Layout fluido

<div class="fluid-container"> estrutura flexível de página, min- e max-widths, e sidebar. Ótimo para aplicações e página de documentações.

<body>
  <div class="fluid-container sidebar-left">
    <div class="fluid-sidebar">
      ...
    </div>
    <div class="fluid-content">
      ...
    </div>
  </div>
</body>

Responsive design Media queries para vários dispositivos e resoluções

Dispositivos suportados

O Bootstrap suporta uma grande variedade de media queries para ajudar a tornar seus projetos mais apropriados em diferentes devices e resoluções. Abaixo o que inclui:

Label Largura do layout Largura da coluna largura das margens (gutter)
Smartphones 480px e inferiores Colunas fluidas, sem larguras fixas
tablets modo retrato 480px até 768px Colunas fluidas, sem larguras fixas
tablets modo paisagem 768px até 940px 44px 20px
Padrão 940px acima 60px 20px
Display maiores 1210px e acima 70px 30px

Template para Media query

O Bootstrap não inclui os media queries automaticamente, mas adicioná-los no projeto é muito fácil e requer o mínimo de configuração:

  1. Use a versão compilada do responsive, bootstrap-responsive.css
  2. Adicionar @import "responsive.less" e recompilar o Bootstrap
  3. Modificar e recompilar responsive.less em separado

Por que não já vem incluso? Nem tudo precisa ser responsive, ao invés disto deixamos a opção de inserir com o mínimo de configuração.

  // phones formato landscape e abaixo
  @media (max-width: 480px) { ... }

  // phones landscape até tablet no formato retrato
  @media (max-width: 768px) { ... }

  // tablet formato retrato ate o landscape e desktop
  @media (min-width: 768px) and (max-width: 940px) { ... }

  // desktop de tamanhos maiores
  @media (min-width: 1210px) { .. }
